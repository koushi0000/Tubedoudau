## 概要

**ツベドウダウ (Tubedoudau)** は  
**YouTubeの動画をダウンロードできるツール**です。

最近ダウンロードサイトで動画保存ができなくなっているため作成しました。

- 解像度：**1080p固定**
- プレイリスト対応
- 音声形式対応

今後の **サポート・アップデート予定はありません。**
