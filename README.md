# ツベドウダウ(Tubedoudau) V1.1
対応プラットフォーム
Windows10,11のみ

# 説明
YouTubeの動画をダウンロードできるツールです。
最近、サイトでダウンロードできなくなっているため作成してみました
今後のサポートやアップデートの予定はありませんし自己責任でお願いします
解像度は1080pのみでdlできます

# 使い方
1.一番上の白い欄に動画を保存したい場所のディレクトリを入力してね
例(C:\Users\Koushi\Documents\ツベドウダウ保存)
2.下の白い欄にダウンロードしたい動画のURLを入れてね
3.下の保存形式を選んでね(Mp3 Mp4 wav)
4.ダウンロードを押すとダウンロードが始まります。
5.ダウンロードが終わるとダウンロード完了と表示されます。

# 仕様的な話
Q.再生リストはダウンロード可能？

A.可能です。再生リストを作成して、限定公開か公開にして
 そのリンクを貼ってあげればダウンロード可能です。
 (動画の量や再生する時間が多いほどダウンロードが時間かかりますが必ずダウンロード完了を出るまでは触らない方がいいです)

Q.ダウンロードするときにツベドウダウの画面が固まって動かない

A.正常な動作です。動画をダウンロードするときは外部のプラグイン的なものでDLしているので
  画面が固まることはよくあります。
  （※すぐ終わるような動画なのに終わらないときは強制終了してやり直してみてください。)

Q.動画をダウンロードしているが、ダウンロード完了とでてファイル破損などが起きる場合がある

A.ストレージの容量が十分にあるかと、メモリが十分に容量あるか確認してください。

Q.どんなPCスペックあればダウンロードが行えますか?

A.CPU:Intel COre I5以上 メモリ:8GB以上 あれば十分なDLは可能だと思います。

Q.今後サポートにはどう考えていますか？

A.サポートやアップデートは終了する考えです。
